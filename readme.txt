== Description ==

Make WP Responsive Table is a lightweight WordPress plugin that enables responsive tables on your website. It allows you to convert existing unresponsive tables into responsive tables without the need for complex coding or additional plugins.

== Installation ==

1. Upload the `make-wp-responsive-table` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
